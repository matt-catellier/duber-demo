# Duber Code Interview

## Onboarding

1. install node 
2. install mongoDB
3. git clone https://github.com/matt-catellier/duber-demo.git
4. cd duber-demo
5. npm install 
6. mongod (start mongoDB)
7. create a database called duber (i.e.'mongodb://localhost:27017/duber')
5. npm start
6. http://localhost:8080/
